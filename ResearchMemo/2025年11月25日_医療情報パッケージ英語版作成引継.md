## SLS確認後の作成フロー

### 🗓️ タスク実施タイミング

**前提条件**:
- SLS（乾燥肌状態）確認完了
- サレックス軟膏・ヘパリン類似物質油性クリームの持参可否判断完了
- 処方薬リストが最終確定

**実施時期**: 12月初旬（出発1週間前）

---

### 📝 作成手順

#### Step 1: 情報収集
```
□ 糖尿病手帳から転記
  - 直近のHbA1c値
  - 直近の血糖値（空腹時・食後）
  - 血圧値
  
□ お薬手帳から転記
  - 処方薬の一般名（英語名）
  - 用量・用法
  
□ 主治医・病院情報
  - 病院名（英語表記）
  - 主治医名
  - 住所・電話番号
  
□ 海外旅行保険情報
  - 保険会社名
  - 証券番号
  - 緊急連絡先
```

#### Step 2: 英訳確認
```
□ 処方薬の英語名確認
  - 薬剤情報提供書
  - または Google 翻訳＋医薬品データベース照合
  
□ 医療用語の英訳確認
  - Type 2 Diabetes Mellitus（2型糖尿病）
  - ADHD（注意欠陥多動性障害）
  - Hypertension（高血圧）等
```

#### Step 3: レイアウト作成
```
□ Word or Googleドキュメントで作成
  - A4サイズ
  - 4つ折り想定（各パネル=A6サイズ相当）
  - フォント：Arial 10-12pt（可読性重視）
  
□ 両面印刷レイアウト
  - 表面：救急・税関用情報
  - 裏面：主治医・特記事項
```

#### Step 4: 印刷・保管
```
□ A4カラー印刷（自宅 or コンビニ）
□ 4つ折りにする
□ ラミネート加工（オプション・100均）
□ メッシュケースに収納
  - 糖尿病手帳の隣
  - お薬手帳のコピーも同梱（オプション）
```

## お薬手帳のコピー同梱（オプション）

### 📖 お薬手帳の英訳不要説

**理由**:
- お薬手帳の写真・コピーをそのまま持参でも可
- 薬剤名・用量は数字で理解可能
- A4用紙が英訳サマリー、お薬手帳が詳細資料として機能

**推奨**:
```
【メッシュケース収納物（医療情報パッケージ）】
├ A4用紙4つ折り（英文医療情報）★NEW
├ お薬手帳のコピー（直近3ヶ月分）★オプション
├ 糖尿病手帳（原本）
├ 処方薬（全量）
├ 三文判＋ミニ朱肉
└ （SLS確認後）サレックス軟膏 or ヘパリン類似物質油性クリーム
```
